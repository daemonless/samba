## Deployment


### Podman Compose

```yaml
services:
  samba:
    image: ghcr.io/daemonless/samba:latest
    container_name: samba
    environment:
      - TZ=UTC
    volumes:
      - "/path/to/containers/samba:/config"
      - "/path/to/containers/samba/shares:/shares"
    ports:
      - 445:445
      - 139:139
    restart: unless-stopped
```
